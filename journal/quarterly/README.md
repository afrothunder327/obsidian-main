# Quarterly Plans

Your quarterly planning space. Plans follow the naming convention: `YYYY-QX` (e.g., `2026-Q1`, `2026-Q2`).

## ✨ Quick Start

### Create a New Quarterly Plan
- Use QuickAdd command: **New Quarterly Plan**
- When prompted, enter the quarter name in format: `2026-Q1`, `2026-Q2`, etc.
- The template will be created in this folder automatically

### Automatic Startup
On Obsidian startup:
1. **Quarterly Plan Check** - If you have a plan for the current quarter (based on today's date), it opens automatically
2. **Missing Plan Reminder** - If no quarterly plan exists for the current quarter, you'll get a notification: "📋 Reminder: Create 2026-QX quarterly plan!"
3. **Daily Note** - Your daily note for today also opens in the background

## 💡 Template Overview
Based on Peter Wildeford's Quarterly Review + Plan Template (V4.1)

**5 Steps (3 hours total):**
1. **Reflection** - Rate 8 life areas, review successes/mistakes
2. **Vision** - Define values and 90-day ideal
3. **Combine & Refine** - Merge insights into 3-5 themes
4. **Rocks & Pebbles** - Set 3 major goals + 2 optional minor ones
5. **Sanity Checking** - Validate time, systems, environment

**Special Features:**
- **Anti-Rocks** - What you WON'T do (critical for focus!)
- **IF-THEN Obstacle Planning** - Pre-plan responses to obstacles
- **MVE (Minimum Viable Effort)** - For habit-change rocks
- **Systems Review** - Check your processes (tracking, habits, task management, accountability)
